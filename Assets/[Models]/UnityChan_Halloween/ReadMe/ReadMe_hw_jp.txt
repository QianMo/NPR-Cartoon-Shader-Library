README.TXT

タイトル：
　　　　ハロウィン版ユニティちゃんモデル　Ver.1.0.0

作成日：
　　　　2014/10/31

対象：
　　　　Unity4.3.4以上推奨

注意：
　　　本モデルを正常に起動するためには、Windows版もしくはOS X版の
　　　Unityがお手持ちのPCにインストールされている必要があります。
　　　最新のUnityインストーラは、
　　　http://japan.unity3d.com/unity/download/
　　　よりダウンロードしてください。
　　　
　　　ハロウィン版をインポートする前に
　　　オリジナルのユニティちゃんパッケージ
　　　http://unity-chan.com/download/datadownload.html
　　　をインポートしてください。

起動するシーンについて：
　　１：Unityを起動して、新規にプロジェクトを作成し、本unitypackageを
　　　　Projectウィンドウにドロップします。
　　２：本unitypackageの展開が終わったら、Projectウィンドウより、
　　　　Assets/UnityChan/Scenesフォルダを開き、
　　　　Scenesフォルダ内のUnityChan_hw.unityをダブルクリックします。
　　３：プレイボタンでシーンを起動します。

モデルとプレファブについて：
　　１：Assets/UnityChan/Prefabs内に、ダイナミクスの設定が終わった
　　　　ハロウィン版ユニティちゃんモデルのプレファブが入っています。
　　２：Assets/UnityChan/unitychan_hwフォルダ内に入っているのが、
　　　　モデル本体のFBXファイルになります。
　　　　ハロウィン版モデルユニティちゃんのマテリアルは、
　　　　Assets/UnityChan/unitychan_hw/Materials内を参照していますが、
　　　　もしマテリアルがなんらかの理由で壊れた場合には、
　　　　Assets/UnityChan/unitychan_hw/UnityChanShader/Materials_hw
　　　　フォルダの中身を、エクスプローラもしくはファインダから上書き
　　　　コピーするとよいでしょう。
　　　　（この場合には、UnityのProjectウィンドウからコピーしてはいけません）

技術的ドキュメントについて：
　　１：Assets/UnityChan/Documentationフォルダの中に、技術的ドキュメント
　　　　があります。
　　２：Unityエディタ上でユニティちゃんを表示した時にあまり綺麗でない
　　　　場合には、Unityエディタ上のQualitySettingsを調整する必要があります。
　　　　詳しくは、unitychan_shader.pdfの「クオリティセッティングのサンプル」
　　　　を参照してください。
　　３：ご自身のプロジェクトでハロウィン版ユニティちゃんを使う場合、
　　　　ライトやダイナミクスの調整をする必要があることがあります。
　　　　その場合も本フォルダ内のドキュメントの一読をお勧めいたします。

Unity Free版での注意：
　　　プロジェクト内で使用している一部機能やポストエフェクトに、
　　　Unity Pro版でないと正常に利用できない項目があります。
　　　もしこれらの項目をご利用になりたい方は、Unity Proの体験版を
　　　ご利用してください。
